Update an action's method descriptions

After the desired object has been located:

  1. Open an associated JSON file that corresponds with the desired action
  1. Locate the method which you wish to update
    • This is an example of the put method for the %7DBID%7D action associated with the proj object

      1. The operationID corresponds to system information and should not be edited
      1. The summary field is used as a title section for the corresponding method
      1. The description field is used to provide additional information about the corresponding method
      1. The tags filed is used to create any additional associations that might be useful in the documentation, primarily for search purposes.

Update a method's parameter descriptions

  1. Find the parameter section of a method
    • This image shows the parameters section of the put method for the %7DBID%7D action associated with the proj object. It contains 2 parameters, the query-updates parameter and the query-BCCompletionState parameter.
      1. The $ref references located on lines 48-53 are references to system file paths and should not be modified for documentation purposes.
      1. The query-updates parameter is located on lines 55-65.
        • Note: the query-updates parameter is a system resource and will not be readily accessible in the developer.workfront UI.
      1. The query-BCCompletionState parameter is located on lines 66-73.
        1. the in field indicates which collection the parameter belongs to and should not be modified for documentation purposes.
        1. the name field is the name of the parameter and should not be modified for documentation purposes.
        1. the description field is the parameter's description and is the summary description that will display in the developer.workfront.
        1. the schema field indicates where the parameter's definition is located in the swagger files and should not be modified for documentation purposes.
          • This reference will tell the RapiDoc UI where it can obtain additional information about the BCCompletionState, like what its possible values are.
