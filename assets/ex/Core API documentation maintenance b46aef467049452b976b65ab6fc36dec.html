Core API documentation maintenance

This documentation is a comprehensive guide to all of the processes involved with updating Workfront's Core API (Redrock API) Documentation and the developer.workfront website. It assumes that users are not familiar with git, command-line operating systems, coding, and other concepts that aren't considered a normal part of the technical-writer skillset at Adobe-Workfront.

These articles are organized in a step-by-step order that describes in detail—or how to obtain—all of the software, credentials, and information that is necessary to maintain all of the API documentation processes. However, each article also describes individual processes that can be used as quick-reference guides for later review.
